Bang, Echo!

Usage: !echo [options] <message...>

This command takes the inputed text and sends it back as a chat message. It will evaluate any roll parameters that you pass to it and output its value.

Example usage:

!echo -a "John Doe" John Doe has a strength of @abilities.str.value

Options:

  • -h, --help: Display's this message
  • -a, --actor: Provide the name of an actor to get data from. If this option is not provided, the command will attempt to get the roll data from the currently selected actor. If no actor can be found, then this help message will appear.
  • -i, --item: The name of an item that this actor owns. This will allow Bang, Echo! to output any roll parameters associtated with that item. e.g. !echo -i "Laser rifle, azimuth" This rifle has @item.capacity.value shots out of @item.capacity.max left.
